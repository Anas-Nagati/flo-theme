# FLO Website

## Project Description
This is a WordPress-based website designed for a marketing and software house. The website was built using **Elementor Page Builder** to create a visually appealing, responsive, and user-friendly layout. It also integrates various plugins for optimization, security, and contact form functionality.
List of plugins used: 
- Elementor
- YoastSelect 
- Contact Form 7 
- Speed Optimizer 
- Wordfence Security

## Installation and Setup Instructions

To install it locally contact me for the full backup.

### Prerequisites
- WordPress 6.7.1 (latest version)
- PHP 7.4 or higher
- MySQL 5.6 or higher
- A web server such as Apache or Nginx
- A local development environment like XAMPP, WAMP, or Local by Flywheel
